Object Oriented Programming (OOP)

OOP is style of programming that involves separating the code into objects that have properties and methods.
It is often used to model representations of objects in the real world. It has three concepts:Encapsulation, Polymorphism, and Inheritance

Encapsulation

Involves keeping all the programming logic inside an object and making methods available to implement
the functionality whitout the outside world needing to know its done

Polymorphism

When various objects can share the same method, but also have the ability to override shared methods with a more specific implementations

Inheritance

We can take an object that already exists and inherit all its properties and methods, then improve on its functionality by
adding new properties and methods
