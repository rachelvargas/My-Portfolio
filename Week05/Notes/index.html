Testing and Debugging

Errors

Are usually caused by:
System Error:When there's a problem with the system or external devices with which the program is interating
Programmer Error: When the program conatains incorrect syntax or faculty logic, it should be as simple as a typo
User Error:When the user has entered data incorrectly which the origram us unable to handle.

Exceptions

Are errors that produce a return value that can be used by the program to deal with error

Warnings

Occur if there's an error in the code that isn't enough to cause the program to crash,
but the program will continue to run after a warning

Debugging

Is the process of finding out where bugs occur in the code and then dealing with them
Debugging Tools allows us to set breakpoints in our code that will pause it at certain
points. It is very useful when trying to track down bugs
