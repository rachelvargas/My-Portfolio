Section Tag

Section tag fill the screen all the way across.

Since the section tag requieres the full width of the page, the buttons can't be on the same line as the two inputs


When inputs

When we want to get the user's input from any 'input' or 'textarea' tags, we use 'value'.

When we want to show something to a user, we use a 'section' and 'innerHTML'

If we are dealing with an input' or 'textarea' use 'value'; for 'section' use innerHTML
